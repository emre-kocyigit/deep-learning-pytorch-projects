# Deep Learning with PyTorch

A collection of deep learning projects built with PyTorch, demonstrating practical skills in model design, training, evaluation, and deployment.

## Projects

| # | Project | Description | Key Concepts |
|---|---------|-------------|--------------|
| 1 | [CIFAR-10 CNN Comparison](./CIFAR'10 CNN Comparison/) | Train and compare a basic vs improved CNN on CIFAR-10 | CNNs, BatchNorm, Dropout, Model Evaluation |

> More projects coming soon — covering topics like transfer learning, GANs, NLP, and more.

## Tech Stack

- **Framework:** PyTorch
- **Environment:** Google Colab (GPU)
- **Evaluation:** scikit-learn, matplotlib, seaborn

## About

This repository includes PyTorch projects with a complete pipeline — from data loading to evaluation — with clean, well-commented code and visualizations of results.

## Getting Started

Each project folder contains its own README with setup instructions. Most notebooks are designed to run directly in Google Colab with zero setup.
